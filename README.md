Pizza Sales Analytics (SQL)
Project Overview

This project focuses on analyzing pizza sales data using SQL to uncover business insights such as revenue trends, top-performing products, and peak order times.

Tools & Technologies

SQL (MySQL)

Data Analysis

KPI Metrics

Business Intelligence Concepts

Key Business Questions Answered

What are the total revenue and average order value?

Which pizzas and categories generate the most sales?

What time and day have the highest order volume?

How do sales trends vary monthly and weekly?

Key Insights

Identified top-selling pizzas contributing maximum revenue.

Discovered peak ordering hours and high-demand days.

Highlighted category-wise performance to support inventory planning.

Outcome

Prepared clean, structured datasets and insights suitable for dashboarding and data-driven decision-making.
